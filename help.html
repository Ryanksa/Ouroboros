About

Ouroboros is a writing tool that pushes for forward thinking. Notes are comprised of lines of text entered by the user. Previously entered lines can only be changed through edit mode (see commands), and new lines simply append to the end of the note.

Moreover, notes can be saved to the local database, and Ouroboros serves as a command center for retriving and editing all the saved notes.

There are more features in Ouroboros to explore, such as grouping and searching the saved notes. Read on for more specific instructions.



How to Use

Ouroboros seeks input from the user with "\>", where the user can type any text to enter into the current note, or enter a command to interact with the system.

When a note is saved, users can set a grouping on it, which will be displayed by "[grouping}".

Additionally, saved notes can be opened and/or edited, which the input seeker will be changed to "\note>" to indicate that note is currently opened.

Users can also peek at saved notes' content without opening them, or search a specific text pattern through all the saved notes.

See the Command section below to see everything a user can do with Ouroboros.



Commands

@exit: exit this shell

@curr: switch console over to the current note

@list: [groupingName}: list all notes, under a certain grouping name if included

@drop noteName [groupingName}: remove notes and grouping of notes as specified. If nothing is specified remove all notes

@save noteName [groupingName}: save the current note into the specified notes and grouping of notes

@setgrp [groupingName} noteName: set the grouping for the specified notes

@rename oldNoteName newNoteName: rename oldNoteName to newNoteName

@open noteName: opens the specified note to curr, saving back to the opened note will replace the content

@edit noteName: edit the given or current (if none was given) note in edit mode

@query [groupingName} 'pattern': queries and open all notes, under grouping if specified, with the given pattern

@peek noteName [groupingName}: peek at the specified notes and groupings of notes
