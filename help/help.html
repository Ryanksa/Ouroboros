About

Ouroboros is a writing tool that pushes for forward thinking. Unlike a standard text editor, entering new text is more accessible than editing entered text.

Moreover, notes can be saved to the local database, and Ouroboros serves as a command center for retriving and editing all the saved notes.


How to Use

Ouroboros seeks input from the user with "\>", where the user can type any text to enter into the current note.

Alternatively, users can enter a command, by typing @command, to interact with the system.

When a note is saved, users can set a grouping on it, which will be displayed by "[grouping}".

Saved notes can be opened and/or edited, which the input seeker will be changed to "\note>" to indicate that note is currently opened.

Users can also peek at saved notes' content without opening them, or search a specific text pattern through all the saved notes.

See the Commands section below to see everything a user can do with Ouroboros.


Commands

@exit: exit Ouroboros

@curr: switch console over to display the current note's content

@clear: erases the current note's content

@list [grouping}: list all notes, under a certain grouping if included

@drop note/[grouping}: remove notes and grouping of notes as specified. If nothing is specified remove all notes

@save note/[grouping}: save the current note into the specified notes and grouping of notes

@setgrp [grouping} note: set the grouping for the specified notes

@rename oldNoteName newNoteName: rename oldNoteName to newNoteName

@open note: opens the specified note to curr, @save will directly save this opened note

@edit note: edit the given or current (if none was given) note in edit mode

@query [grouping} 'pattern': peek all notes, under grouping if specified, with the given pattern

@peek note [grouping}: peek at the content of specified notes and groupings of notes
